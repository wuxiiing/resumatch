# 简配 · 求职知识库（RAG 语料）

> 用户提供的求职领域知识，**尽量原样保留**，作为「简配 / 求职军师」的 RAG 语料。
> 之后用于:① 增强军师研判与对话(引用真实求职方法论);② 「职业规划」板块(专业/背景 → 适合岗位);③ 「修改简历」时的 STAR 等方法论支撑。

## 文档
1. `01-求职攻略-心态简历面试.md` —— 学姐版攻略:心态建设、STAR 简历法、面试攻略、秋招/春招时间线、企业类型特点。
2. `02-求职四大误区.md` —— 专业对口执念 / JD 没写专业就不投 / 不知道投什么 / 僵化关键词搜索。**与简配内核一致(看穿 JD、不被专业绑死)。**
3. `03-专业适合岗位库.md` —— 数十个专业 → 适合岗位 / 典型公司的对照库(按专业分节,便于按专业检索)。
