## 关键决策（别推翻）
- **节节高 = 岗位定位**(目标/跳板/该绕开,后端 `decideVerdict` 定),**不是打分**;匹配度 = 从 `gaps` 派生的"命中/部分/缺"覆盖。**不要重新引入 0-100 分。**
